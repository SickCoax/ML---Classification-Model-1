# ML Classification Model

This project is a Machine Learning classification model built using Python which can predict whether the person is dead or alive in titanic ship based on there status and class.

## Features
- Data Preprocessing
- Handling missing values
- Feature scaling (numerical features) / Encoding (Catagorial features)
- Model Training
- Model Evaluation

## Library Used
- numpy
- pandas
- matplotlib
- scikit-learn

## Model Used
- CLassification Model (Logistic Regression)
